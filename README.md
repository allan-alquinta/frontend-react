# Frontend React – Sistema de Gestión Eléctrica

Frontend desarrollado en React para el Sistema de Gestión Eléctrica.

## Descripción
Este proyecto corresponde a la interfaz de usuario del sistema, encargada de la
autenticación, navegación y visualización de información proveniente de una API REST
desarrollada con Django REST Framework.

El frontend se encuentra en desarrollo continuo y actualmente permite consumir datos
desde la API mediante operaciones GET.

## Funcionalidades implementadas
- Login y logout de usuarios
- Rutas protegidas
- Dashboard principal
- Navegación mediante sidebar
- Visualización de Clientes
- Visualización de Contratos
- Visualización de Pagos
- Visualización de Medidores
- Consumo de API REST (GET)

## Tecnologías utilizadas
- React
- Vite
- JavaScript
- CSS
- Axios / Fetch API

## Integración con Backend
Este frontend se comunica con una API REST desarrollada en Django REST Framework,
encargada de la lógica de negocio y persistencia de datos.

## Estado del proyecto
Proyecto en desarrollo.
Se encuentra en proceso de incorporación de funcionalidades POST para la creación
de registros desde el frontend.
